# Feline Inspiration Covenant

A compact museum-style fortune draw page themed around curated cat portraits.

## Live Preview

After GitHub Pages is enabled for this repository, the site URL will be:

`https://jayuedu.github.io/cat-fortune/`

## Project Files

- `index.html`: single-file page with layout, styles, SVG artwork, and interaction

## Local Preview

You can open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit:

`http://localhost:8000/`
